HTML → structure  
CSS → design  
JS → logic  
React → architecture

HTML: HTML is used to create the structure of a web page.

CSS: CSS is used to style and design a web page.

JavaScript: JavaScript is used to add logic and interactivity to a web page.

React: React is a JavaScript library used to build user interfaces using components.

Java: Java is an object-oriented programming language used to build backend and enterprise applications.


Topics that are similar in both
Concept	Java	JavaScript
Variables	int a = 10	let a = 10
Conditions	if, else	if, else
Loops	for, while	for, while
Functions	methods	functions
Arrays	int[] arr	let arr = []
Objects	classes	objects
OOP	strong	flexible
Logic	same	same

Logic is universal. Only syntax changes.
